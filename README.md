# Assistente Cartório

Aplicativo para Windows que auxilia rotinas administrativas de cartório, incluindo preparação de reuniões no Teams, criação de e-mails de ofício, certidões, contatos e reiterações.

## Download

Baixe sempre a versão mais recente na página:

https://github.com/contadescartavel26/assistente-cartorio/releases/latest

Na seção **Assets**, clique em `Assistente_Cartorio.exe`.

## Como usar

1. Baixe `Assistente_Cartorio.exe`.
2. Salve o arquivo em uma pasta na qual você tenha permissão para gravar.
3. Abra o executável.

O programa cria e mantém as configurações do usuário na mesma pasta do executável. Por isso, evite executá-lo diretamente da pasta de Downloads temporários ou de um local protegido do Windows.

## Atualizações

O programa consulta automaticamente a última versão publicada no GitHub ao iniciar. Quando houver uma atualização, ele mostra as versões instalada e disponível e oferece o link direto para download.

Também é possível fazer a consulta manual na aba **Config. & Logs**, pelo botão **Verificar atualizações**.

O programa não instala a atualização sozinho: o usuário confirma e baixa o novo executável.
